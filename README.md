# big-white-whale-travel-guide

## **大白鲸旅行攻略**

- 项目简介：一款非常优秀的旅游攻略方面的微信小程序

- 项目已经上线 有疑问可以提Issues 我看到会及时跟进的

## **目录结构：**
- images — 存放项目图片
- pages — 存放项目页面相关文件
- style — 存放独立wxss样式文件，可import引入
- utils — 存放utils文件，可require引入
